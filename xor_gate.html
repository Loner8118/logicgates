How the XOR Gate Works:

The XOR (exclusive OR) gate is a digital logic gate that outputs true (1) only if exactly one of its inputs is true (1). If both inputs are false (0) or both are true (1), the output is false (0). For example:

  • If A = 0 and B = 0, then A XOR B = 0
  • If A = 0 and B = 1, then A XOR B = 1
  • If A = 1 and B = 0, then A XOR B = 1
  • If A = 1 and B = 1, then A XOR B = 0
