NAND Gate

Output:

How the NAND Gate Works:

The NAND gate is a digital logic gate that outputs false (0) only if both of its inputs are true (1). In all other cases, the output is true (1). It is essentially the inverse of the AND gate. For example:

  • If A = 0 and B = 0, then A NAND B = 1
  • If A = 0 and B = 1, then A NAND B = 1
  • If A = 1 and B = 0, then A NAND B = 1
  • If A = 1 and B = 1, then A NAND B = 0
